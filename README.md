# keyploy-task-4
A webapp built using HTML, CSS and Nodejs to serve custom built Anime RESTful API.

## To setup follow these steps:
### Frontend -
```sh
cd frontend
node app.js
```

### Backend -
```sh
cd backend
node index.js
```

Now the app is running successfully, to access it http://localhost:3000.
